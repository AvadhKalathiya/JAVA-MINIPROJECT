🎮 Java Mini Project – Number Guessing Game

An interactive Number Guessing Game built using Java.
The program generates a random number between 1 and 100, and the player must guess it with hints provided after each attempt.

📌 Project Overview
The Number Guessing Game is a beginner-friendly Java console application that demonstrates:
Random number generation
Looping concepts
Conditional statements
User input handling
Attempt tracking logic
The game continues until the correct number is guessed.

✨ Features
🎲 Random number generation using Random
🔁 Loop until correct guess
📊 Tracks number of attempts
🔼 Hints: "Too High" / "Too Low"
🎉 Success message on correct guess
💻 Console-based interactive game

🛠️ Tech Stack
Language: Java
Concepts Used:
Scanner class
Random class
while loop
if-else statements
Variables & counters
