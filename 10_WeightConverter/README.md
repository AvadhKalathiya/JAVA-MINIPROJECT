⚖️ Weight Converter – Java

A simple console-based Weight Converter built using Java.
This program allows users to convert weight between Kilograms, Grams, and Pounds using a menu-driven interface.

📖 Project Overview
The Weight Converter is a beginner-friendly Java application that demonstrates:
User input handling
Switch-case statements
Basic arithmetic operations
Unit conversion logic
The user selects a conversion type, enters a weight value, and the program displays the converted result.

🔄 Available Conversions
Kilogram → Gram
Kilogram → Pound
Gram → Kilogram
Pound → Kilogram
