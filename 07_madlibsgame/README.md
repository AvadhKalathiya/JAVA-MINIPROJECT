🎮 Java Mini Project – Mad Libs Game

A simple and interactive Mad Libs Game built using Java.
This console-based application takes user inputs and generates a fun and creative story dynamically.

📌 Project Overview

The Mad Libs Game is a beginner-friendly Java mini project designed to practice:

User input handling using Scanner

String manipulation

Java basic syntax

Console-based program flow

The program asks the user to enter different types of words and automatically creates a humorous sentence.

✨ Features

🎮 Interactive console gameplay

🧠 Dynamic story generation

📝 User input using Scanner class

💻 Beginner-friendly Java project

⚡ Lightweight and easy to run

🛠️ Tech Stack

Language: Java

Concepts Used:

Variables

User Input (Scanner)

Strings

Console Output
