🧮 Smart Math Calculator – Java

A console-based Smart Math Calculator built using Java.
This interactive program allows users to perform multiple mathematical operations using Java's built-in Math class.

📖 Project Overview
The Smart Math Calculator is a menu-driven Java application that performs various mathematical operations like:
Square Root
Power calculation
Maximum & Minimum
Absolute Value
Rounding numbers
Random number generation
The program runs continuously until the user chooses to exit.

✨ Features
📋 Menu-driven interface
🔁 Loop-based continuous execution
🧠 Uses Java Math class functions
🎲 Random number generation (1–100)
❌ Exit option
⚡ Beginner-friendly and interactive


🛠 Technologies Used
Java
Scanner class (user input)
Math class
switch-case
do-while loop
Conditional logic
